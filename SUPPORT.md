# Support Policy

## How to Get Help

- **Documentation**: Read the [README.md](README.md) for usage, setup, and Mock mode instructions.
- **GitHub Issues**: Search open issues before submitting a new issue.
- **Security Issues**: See [SECURITY.md](SECURITY.md) to report security vulnerabilities privately.

## Supported Platforms

WingetUpdater supports:
- Windows 10 (version 1809+); v1.1.0 is manually validated on Windows 10 22H2
- Windows PowerShell 5.1 and PowerShell 7 (pwsh)
- winget 1.4.0 or newer

Windows 11 is not currently supported.
