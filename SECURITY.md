# Security Policy

## Supported Versions

Only the latest published release line of WingetUpdater is supported with security updates. Until v1.1.0 is published, the latest v1.0.x release remains supported. After v1.1.0 publication, v1.0.x will be marked unsupported and linked to the replacement release.

| Version | Supported          |
| ------- | ------------------ |
| v1.1.x  | Supported after v1.1.0 publication |
| v1.0.x  | Supported until v1.1.0 publication |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability in WingetUpdater, please report it privately via [GitHub Private Vulnerability Reporting](https://github.com/Zalesny123/WingetUpdater/security/advisories/new).

When reporting a vulnerability, please include:
- A description of the issue and its potential impact
- Detailed steps or proof-of-concept to reproduce the behavior
- Any suggested mitigations or fixes

You will receive an initial response acknowledging receipt within 48 hours.
